# Shipwreck & Fishing Database

Historic shipwreck data and fishing spots for the Atlantic, Caribbean, Gulf of Mexico, and Americas.

## Live Maps

- **Atlantic & Americas Shipwrecks**: Interactive map with pre-1800 treasure ship focus
- **Florida Wrecks**: Detailed Florida coast wrecks and fishing spots
- **Caribbean & South America**: Caribbean basin and South American wrecks

## Database Files

### Shipwrecks
- `shipwreck_database.json` - Main curated database (v3.1) with 30+ treasure ships
- `east_coast_wrecks.json` - US East Coast wrecks
- `gulf_wrecks.json` - Gulf of Mexico wrecks

### Fishing & Reefs
- `fishing_spots.json` - Known fishing locations
- `fishing_spots_enhanced.json` - Enhanced fishing data with details
- `fwc_reefs.json` - Florida Fish & Wildlife reef data
- `osm_fishing.json` - OpenStreetMap fishing data
- `osm_reefs.json` - OpenStreetMap reef data

### Bathymetry
- `gmrt_cocoa.asc` - GMRT bathymetric data for Cocoa Beach area
- `gmrt_cocoa_bathy.tif` - GeoTIFF bathymetry
- `bathy_map.svg` / `bathy_page.html` - Bathymetry visualization

## Map Features

- **Depth information** for each wreck (known or estimated)
- **Uncertainty circles** for unfound/approximate locations
- **Flyout detail panels** with cargo, casualties, sources
- **Pre-1800 treasure era** filter
- **Peru silver route** overlay
